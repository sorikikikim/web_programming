블록 태그와 인라인 태그

블록 태그 사례 : <p>, <h1>, <div>, <ul>
블록 태그는 항상 새 라인에서 시작. 브라우저의 왼쪽 끝에서 오른쪽 끝까지 블록 공간을 차지함.
인라인 태그 사례 : <strong>, <a>, <img>, <span>
반대로 인라인 태그는 블록에 삽입되어 블록 콘텐트의 일부를 표현하는 데 이용됨.

많이 사용하는 블록 태그 <div>

<div>는 <p>와 함께 블록을 구성하기 위해 가장 많이 사용된다. <p>태그는 문단을 만들기 위해 사용되지만, <div>는 특별한 의미 없이 여러 html 태그들을 블록으로 묶는 컨테이너로 이용된다. 예를 들어 <div> 블록 전체에 동일한 CSS 스타일을 적용하거나 자바스크립트로 블록을 하나의 단위처럼 다루고자 할 때이다.

많이 사용하는 블록 태그 <span>

<span> 태그는 텍스트 일부분에 특별한 모양을 주거나, 자바스크립트 코드로 텍스트 일부분을 제어하고자 할 때 사용한다.
